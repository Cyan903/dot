## Usage

Requires [yolk](https://elkowar.github.io/yolk/). To use Wayland, create an empty file at `~/.config/yolk/.wayland`. Themes and settings can be configured in [colors.rhai](colors.rhai) and [yolk.rhai](yolk.rhai).

```sh
$ yolk help
$ yolk sync
```
